# VM-Based-CI-CD-Pipeline-Using-Jenkins-Ansible
VM-Based CI/CD Pipeline Using Jenkins & Ansible
Project: VM Based CI/CD Pipeline using Jenkins & Ansible

Tools Used:
AWS EC2
Jenkins
Maven
SonarQube
JFrog Artifactory
Ansible
Apache Tomcat
Git

Pipeline Flow:
GitHub → Jenkins → Maven Build → SonarQube → JFrog → Ansible → Tomcat Deployment

Description:
This project demonstrates a VM-based CI/CD pipeline where Jenkins automates
the build process and Ansible handles deployment to Tomcat servers.
